\subsection{Descrição Detalhada}
Classe abstrata para representar a inteligencia de um jogador do jogo de Domino. Termos tecnicos do jogo Domino RAXASSAUM:
\begin{DoxyItemize}
\item RANDOM: Calouro. Faz escolhas aleatorias.
\item POPS: Calouro depois de 5 minutos jogando. Joga as pecas com maior peso sem ver nada.
\item Rachar: Jogar nas pontas com maior peso para impedir o proximo jogador de rolar suas buchas pesadas.
\item Bucha: \hyperlink{classPeca}{Peca} com o mesmo numero dos dois lados.
\item Rolar: Jogar uma bucha. 
\end{DoxyItemize}
